Pour exécuter l'application mobile pour la gestion des commandes de déjeuner dans une entreprise, il suffit de procéder les deux
étapes suivantes:

1er étapes: lancer le strapi application il faut:
- se placer sous le repertoire myRestaurantCMS
- et exécuter la commande
	strapi start

2eme étape: lancer l'application Ionic myFastFood il faut:
- se placer sous le repertoire myFastFood
- et exécuter la commande 
	ionic cordova run browser 

NB : Il n'y a pas de dossier node_modules dans les deux projets myRestaurantCMS et myFastFood
